# zBox appliance

Based on Debian GNU Linux

## Purpose

My personal all-in-one VM for testing & developement

- Fancy zsh prompt shell (oh-my-zsh/posh/custom theme)
- Pre-configured apt sources list for docker,kubernetes,powershell,hashicorp
- Various misc tools
